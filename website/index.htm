An alternative binary encoded vector graphics format.

Why another format?

SVG is horribly complex and an overkill for most projects. The specification includes way too many edge cases so implementing a new SVG renderer will always have drawbacks or missing pieces.

TVG tries to be simpler. Less features, but powerful enough to cover 90% of use cases.

Features

  • Compact binary encoding (XX% size of a comparable SVG file)
  • Comprehensible specification
  • Paths, lines, rectangles
  • Two-point gradient support
  • Configurable precision for smaller files

Downloads
